# 安装说明

## 电脑端安装

1. 下载 `aiob-board-study-planner-major-workbench-20260510.zip`。
2. 解压压缩包。
3. 把解压后的插件文件夹放到：

```text
你的 Obsidian 库/.obsidian/plugins/aiob-board/
```

4. 打开 Obsidian，进入：设置 -> 第三方插件。
5. 关闭安全模式或开启社区插件。
6. 启用 `Aiob Board`。
7. 回到插件页面后按 `Ctrl + R` 刷新。

## 手机端安装

1. 手机下载这个仓库里的 zip 插件包。
2. 解压后，把插件文件夹放进手机 Obsidian 库：

```text
.obsidian/plugins/aiob-board/
```

3. 确认文件夹里有：

```text
manifest.json
main.js
styles.css
```

4. 打开 Obsidian 手机端，开启第三方插件并启用 Aiob Board。

## 同步提醒

如果使用 Obsidian Sync、Syncthing、坚果云、OneDrive 或 iCloud，请确认 `.obsidian/plugins/` 没有被排除同步。
