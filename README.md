# Aiob Board Study Planner

这是一个自用版 Obsidian 插件包，基于 Aiob Board 改造成学习、知识规划和生活规划工作台。

## 当前功能

- 大项工作台：把学习、生活、项目拆成可管理的大方向。
- 自由分类：分类可以新增、改名、删除、调整顺序，并决定是否参与今日计划。
- 大项专属页面：点击大项后，可以直接写这个科目或项目今天要做什么。
- 今日任务、周目标、复习、资料箱、章节地图、AI 助手入口。
- 适配电脑和手机端手动安装。

## 下载和安装

1. 在仓库里下载插件压缩包：`aiob-board-study-planner-major-workbench-20260510.zip`。
2. 解压后，把里面的插件文件夹放到你的 Obsidian 库：

```text
你的库/.obsidian/plugins/aiob-board/
```

3. 确认这个文件夹里至少有：

```text
manifest.json
main.js
styles.css
```

4. 打开 Obsidian：设置 -> 第三方插件 -> 关闭安全模式/开启社区插件 -> 启用 Aiob Board。

## 手机同步提醒

如果电脑和手机使用同步工具，请确认 `.obsidian/plugins/aiob-board/` 也被同步。很多同步工具默认只同步笔记，不同步隐藏配置文件夹。
